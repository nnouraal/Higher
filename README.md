# HIGHER Game 🃏

A fun and simple two-player card game built with SwiftUI. Challenge the CPU and see who gets the HIGHER card!

## 🎮 About the Game

The player and CPU each draw a random card. The player with the higher card wins the round and earns 1 point. Simple, fast, and addictive!

- Built with SwiftUI
- Clean and playful UI
- Perfect for beginners learning iOS development

## 🛠️ Technologies Used

- SwiftUI
- Xcode
- iOS Development

## 📸 Features

- Random card drawing (cards 0–10)
- Real-time score tracking
- Custom logo and reveal button
- Pastel pink background
- Responsive layout using `ZStack`, `VStack`, and `HStack`

## 🚀 How to Run

1. Clone this repository
2. Open `HIGHER Game.xcodeproj` in Xcode
3. Run on simulator or real device


## 🕹️ How to Play

1. Tap the **reveal button** to draw cards
2. Compare card values:
   - Higher card → +1 point to that player
   - Equal cards → no points
3. Keep playing and see who wins!
